关于 AI 创客指南 ​

我是谁 ​

一个在 AI 工具链领域深耕的内容创作者。日常使用 Claude Code、Cursor、HyperFrames 等工具做开发，踩坑无数，总结成文。

这个网站是什么 ​

中文 AI 工具教程的内容荒地。

2026 年，AI 编程和 AI 视频领域每天都有新工具、新框架、新范式出现。但中文深度教程极度稀缺——HyperFrames 开源 3 个月，中文系统教程几乎为零；Agent Skills 概念火了半年，除了官方文档，能看的中文内容屈指可数。

这个网站试图填补这个空缺。每篇文章都是实战验证过的深度教程，不是机翻官方文档，不是泛泛的介绍文。

写作原则 ​

  1. 只写自己跑通过的工具：每行代码都亲自执行过
  2. 不只告诉你"怎么做"，更告诉"为什么这样做"：理解原理比复制粘贴有用
  3. 持续更新：工具在变，文章也跟着变
  4. 中文优先：为中文开发者而写，不是翻译英文博客

内容计划 ​

  • HyperFrames 系列教程（第一篇已发布）
  • Agent Skills 实战系列（第一篇已发布）
  • AI 视频工具链对比测评
  • 工业化视频生产流水线搭建
  • Claude Code 深度使用技巧
