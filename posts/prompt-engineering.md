## 为什么你的提示词总是"差一点"？

你写："一个赛博朋克风格的街道，霓虹灯，下雨。"

AI 生成：一个亮堂堂的、不下雨的、不知道什么风格的街道。

问题在哪？**你给的是"印象"，AI 需要的是"参数"。**

提示词工程就是把"印象"翻译成"参数"的学科。这不是艺术，是工程——有输入规范、有质量标准、有可复现的方法。

## 核心框架：五层结构

每一条合格的提示词必须按以下五层组织：

```
[主体描述层] → [环境与场景层] → [光线设定层] → [技术摄影层] → [风格与美学层]
```

### 第一层：主体描述

回答一个问题：**画面里最重要的东西是什么？**

```diff
- ❌ 一个老人
- ✅ 一位约75岁的东亚男性，灰白短发，面部有深皱纹（特别是眉心竖纹和法令纹），
     穿着藏青色棉麻中山装，左手拄一根竹节拐杖，右手自然垂放身侧。
     身高约170cm，体型偏瘦。站在画面正中央，占画面高度约60%。
```

**关键要素**：年龄、性别、体型、穿着（颜色+/#hex）、姿态、表情、空间位置。

### 第二层：环境与场景

回答：**这个人/物在什么地方？**

```diff
- ❌ 在老街上
- ✅ 江南水乡石板老街，两侧是明清风格的木构建筑。路面有积水，
     倒映灯笼光线。远景有一座石拱桥，桥上有3-4个模糊的人影。
     空气中可见薄雾。时间是黄昏。
```

**关键要素**：场景类型、空间尺度、背景处理（清晰/虚化）、大气条件、时间。

### 第三层：光线设定

这是**最常被忽略但最重要**的一层。光线决定画面的质感和情绪。

```diff
- ❌ 暖光
- ✅ 主光源：画面右侧斜上方45°的暖色路灯 (#ff9900)，照度约200 lux。
     辅助光：左侧建筑窗户透出的暖黄光 (#ffcc66)，照度约80 lux。
     环境光：暮色天光从画面顶部漫射，色温约6000K，照度约50 lux。
     光质：主光为柔光（透过灯笼纸的漫射），阴影边缘柔和。
```

**光线参数表**：

| 属性 | 需要指定 |
|------|------|
| 光源类型 | 阳光/月光/灯光/自发光 |
| 色温 | K值或#hex |
| 方向 | 角度+方位（如"45°左侧上方"） |
| 光质 | 硬光（锐利阴影）/柔光（柔和阴影）/漫射 |
| 照度比 | 主光:辅光:环境光 = X:Y:Z |

### 第四层：技术摄影

```diff
- ❌ 拍好一点
- ✅ 机位：平视，高度约120cm（腰部水平）。等效焦距85mm（中长焦）。
     景深：浅景深，主体清晰，背景虚化（光圈f/2.8效果）。
     构图：三分法，主体在右1/3处，视线方向留白。
     曝光：中调偏暗，暗部有细节。
```

### 第五层：风格与美学

```diff
- ❌ 要有电影感
- ✅ 视觉风格：超写实渲染，PBR材质。肤质保留微毛孔和纹理，
     不磨皮。色彩基调：暖琥珀+冷蓝的冷暖对峙（橙青调色）。
     后期处理：无胶片颗粒，保留传感器原生微噪点。
     白平衡：~5200K（略低于主光，肤色微偏暖）。
```

## 翻车案例诊断

### 案例 1："AI 感"太重

**原始提示词**：
```
一个漂亮的女人，在咖啡店里，阳光从窗户照进来，温馨的氛围
```

**诊断**：全是抽象形容词（"漂亮""温馨"），AI 只能用训练数据中的"平均值"来填充，结果就是千篇一律的 AI 脸。

**修复**：
```
一位约28岁的女性，面部不对称（左眉比右眉高约2mm，左眼略大于右眼），
鼻梁右侧有一个直径约1mm的浅褐色痣。皮肤T区偏油有微光泽，两颊偏干。
发丝有几根不规则的乱发。坐在咖啡店靠窗位置，右手握白色陶瓷咖啡杯。
上午10点的阳光从左侧窗户以约60°角射入，在她的右侧脸和桌面投下暖色光斑。
```

### 案例 2：角色不统一

**问题**：用同一段提示词，每次生成的角色像不同的人。

**修复**：建立**角色锚点描述**，每次生成都放在提示词最前面：

```
角色锚点-张明：
脸型：方脸，下颌角约110°，颧骨微凸。肤色：小麦色(#c8a882)，比平均偏深。
眼睛：单眼皮，眼裂较窄，左眼外眼角微微上挑。眉毛：浓黑，眉尾略下垂。
鼻子：鼻梁挺直，鼻头略圆。嘴唇：下唇比上唇厚约1.5倍。发型：黑色短发，三七分。
身高体重：178cm，约75kg，肩宽约48cm。
```

把这个角色锚点贴在每个分镜提示词前，AI 就有了同一张脸。

### 案例 3：场景空洞

**问题**："一个科幻城市"——生成的画面总感觉"缺东西"。

**修复**：把"印象"替换成可验证的物体清单：

```diff
- ❌ 一个科幻城市
- ✅ 画面中包含：3座锥形摩天楼（高度递减）、12个悬浮飞行器（不同距离）、
     8块全息广告牌（大小不一）、1条多层高架路（5层）、
     地面有约20个行人（远景剪影）、空中轨道上有2列透明列车。
```

## 视频提示词的额外维度

视频提示词 = 图片五层结构 + 以下四项：

### 1. 镜头运动

| 类型 | 写法示例 |
|------|------|
| 固定机位 | "固定机位，画面运动仅来自画面内物体的位移" |
| 推轨 | "缓慢推轨(dolly in)，速度约每秒画面宽度的2%，持续8秒" |
| 横移 | "右向横移，速度约0.5m/s，镜头中心始终跟随主角" |
| 摇镜 | "从左上角向右下角缓慢摇镜(pan)，角度变化约15°，持续6秒" |

### 2. 画面内动态

```
画面内动态元素：
- 人物的头发被微风轻轻吹动（风速约2m/s，方向左→右）
- 桌面上的纸张边缘微微翘起又落下
- 窗外树叶以约1Hz频率摇晃
- 咖啡杯中的热气以约0.5cm/s速度上升并扩散
```

### 3. 首尾帧过渡

```
首帧(0s)：主角正面站立，表情平静，背景为模糊的城市街道。
尾帧(10s)：主角转身约90°走向画面深处，背影渐小，城市街道变清晰。
过渡方式：推轨跟随主角转身，景深从浅变深，背景从虚化渐变清晰。
```

### 4. 音频提示词（部分工具支持，具体以官方文档为准）

```
BGM：轻柔钢琴独奏，BPM 70，大调，情绪平和中带一丝淡淡忧伤。
音效：0s-2s 风声（低频），5s 远处汽车鸣笛，8s 主角脚步声（渐弱）。
```

## 工业化模板体系

### 模板 1：电商产品展示

```
[产品名称]在[场景]中。[光线]从[方向]照射，产品表面呈现[材质质感]。
镜头从[角度]以[运镜方式]运动，持续[时长]秒。背景[描述]。
整体风格[风格类型]，色彩基调[色调]。产品占画面[比例]。
```

### 模板 2：文旅宣传

```
[地点名称]的[标志性景观]，[时间][天气条件]。[光线条件]。
前景:[元素]，中景:[元素]，远景:[元素]。
[运镜方式]从[起点]到[终点]，持续[时长]秒。
色彩[主色调]，饱和度约[数值]%。
```

### 模板 3：知识科普

```
[知识点]的视觉化呈现。主体是[抽象概念]的[具象化表达]。
[辅助元素]环绕主体，以[动画方式]运动。背景为[颜色/渐变]。
文字[内容]出现在[位置]，字体[类型]，[入场方式]。
整体风格简洁现代，色调[描述]。
```

## 提示词调试清单

每次生成不满意时，按顺序排查：

| # | 检查项 | 问题 | 修复 |
|:--:|------|------|------|
| 1 | 主体明确吗？ | 画面不知道拍什么 | 第一层加细节 |
| 2 | 光线指定了吗？ | 画面没有质感 | 第三层写光源类型+方向 |
| 3 | 焦距指定了吗？ | 构图没有重点 | 第四层写焦距+景深 |
| 4 | 风格指定了吗？ | 画面风格飘忽 | 第五层明确写实/插画/渲染 |
| 5 | 有具体数值吗？ | "好看一点"太主观 | 加精确数值（mm/cm/°/K值） |
| 6 | 禁止词写了吗？ | AI 乱加东西 | 明确说"不含XX""禁止XX" |
| 7 | 有参考负例吗？ | 修正后仍在错 | 给一个"不是这样"的例子 |
